# 🛡️ Bulletproof Data Anonymizer

A free, local, zero-dependency Claude Skill that instantly scrubs Personally Identifiable Information (PII) from text.

## 🚀 How It Works
It intercepts the user's raw text and runs it through strict regex patterns in Python to replace emails, phone numbers, Social Security Numbers, and IP addresses with safe `[REDACTED]` tags. This ensures sensitive data is never processed into the AI's core conversation memory.

## 🛠️ Usage
1. Upload the zip to Claude Skills.
2. Trigger it by saying: *"Anonymize this text: [paste text]"*
