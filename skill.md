---
name: Bulletproof Data Anonymizer
description: Scans and scrubs Personally Identifiable Information (PII) like emails, phone numbers, and SSNs from text.

# Instructions

1. When the user asks to anonymize, scrub, or redact a block of text, do not process the text yourself.
2. Pass the exact, raw text provided by the user as a single string argument to `anonymizer.py`.
3. Output the exact scrubbed text generated by the script back to the user without altering it.
